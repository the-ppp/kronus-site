Encontre respostas às perguntas mais frequentes ou entre em contacto diretamente connosco.

Como funciona o registo de tempo no Kronus?

O Kronus permite-lhe registar as suas atividades em blocos de 15 minutos ao longo do dia. No final do dia, obterá uma visão clara de como distribuiu o seu tempo.

Os meus dados são guardados de forma segura?

Sim. Os seus dados são armazenados localmente no seu dispositivo. Consulte a nossa Política de Privacidade para mais detalhes.

O Kronus está disponível para Android?

Atualmente o Kronus está disponível para iOS. Uma versão para Android está em desenvolvimento — registe-se na nossa lista de espera para ser notificado.

Como posso apagar os meus dados?

Pode apagar todos os seus dados a partir das definições da app, em Definições → Dados → Apagar todos os dados. Esta ação é irreversível.
